# book-scanner
Book scanner appliaction for scanning books and saving as a single multipage TIFF file after a little bit of editing.

Pretty straightforward application which allows you scan multiple pages of a book, with a ability to crop selected area (especially those black one you get while scanning books). Gathered images can be then save a single files or combined multipage TIFF file.
PDF support isn't there unfortunatelly.

## Usage instructions
1. Select scanner from menu (the default one is usually selected)
2. Start operation by clicking `New` button in `operations` area.
3. Start scanning
